# Premium Animated Glassmorphic Login and Registration Page

This is a modern, responsive, and animated login and registration interface built using vanilla HTML5, CSS3, and Javascript. 

## Features
- **Moving Gradient Ambiance**: CSS-animated drifting ambient blobs that respond to dark and light mode.
- **Glassmorphic Card Design**: Transparent card structure utilizing `backdrop-filter` and fine glowing borders.
- **Form Tab Transitions**: Smooth custom sliding tab indicator and layout transition between Sign In and Sign Up views.
- **Floating Labels**: Fully interactive CSS-driven inputs where labels transition smoothly when focused or containing content.
- **Interactive Password Visibility**: Dual toggles with dynamic show/hide SVG icon transitions.
- **Material Design Ripples**: Javascript-calculated click position ripples on submit actions.
- **Interactive Dynamic Submit Loaders**: Submit actions shift into spinner loaders, then green success states with scale-up checkmark icons.
- **Detailed Form Validation**: Instant client-side validation check on field blur, with animated shake feedbacks on invalid submissions.
- **Responsive Dynamic Toast System**: Layered notification system that slides in dynamically from the corner.
- **Dark & Light Mode Switcher**: Toggles theme styles, automatically saving the user's choice to `localStorage` and checking system preferences.

## File Structure
- `index.html`: Semantic HTML elements, importing Outfits font and Lucide Icons.
- `style.css`: Theme variables, ambient animation patterns, responsive layout grids, and visual tokens.
- `script.js`: Swapping behaviors, eye toggles, toast constructors, form validators, ripple elements, and loaders.

## How to Run
Simply open the `index.html` file in any modern web browser or serve it using a local static file server.
